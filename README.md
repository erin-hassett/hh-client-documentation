# Humbly Hewn Client Documentation

Client-facing documents for Humbly Hewn Photography, hosted via GitHub Pages.

## Document Types

- **Timelines** — Wedding day timelines (`/timeline` in MARVIN)
- **Family Photo Lists** — Family portrait shot lists (coming soon)

## Structure

```
/timelines/scout-alex.html    → erin-hassett.github.io/hh-client-documentation/timelines/scout-alex
/family-lists/scout-alex.html → erin-hassett.github.io/hh-client-documentation/family-lists/scout-alex
```

## Adding a new document

1. Generate HTML using MARVIN skill (`/timeline`, etc.)
2. Copy to appropriate folder with client-friendly filename
3. Commit and push
4. Share URL with client

## GitHub Pages

Enable in repo Settings > Pages > Source: main branch, / (root).
